# Book build instructions

This project contains the book markdown in `book/`. You can build a static site using MkDocs or any static markdown site generator.

Install MkDocs (Windows PowerShell):

pip install mkdocs mkdocs-material

Preview the site:

mkdocs serve -a 127.0.0.1:8000

Then open http://127.0.0.1:8000 in your browser.

Alternatively copy the markdown into any static site generator.
